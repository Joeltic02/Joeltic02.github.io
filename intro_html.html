Salto de línea con "br"

Nos sirve para hacer saltos de líneas que permite que los textos no abunden
tanto y tener más espacio o tambien se utiliza para canciones o poemas

Estilo del texto

Nos sirve para cambiar la forma del texto y también con esto destacar distintas palabras Se hace utiliando las siglas "em" y poniendo la terminación hasta donde quieras "/em"

Lista desordenada

Sirve para hacer una lista para cuando no necesitas que algo vaya antes que lo otro, es decir, sin orden concreto.
Para ello escribimos <"ul">, pones lo que quieres colocar con el comando(<"li">) y se cierra con <"/ul">.

Lista ordenada

Sirve igual que la lista desordenada, pero esta utiliza un orden (1. 2. 3. etc).
Este comando es similar al anterior pero tenemos que escribir en vez de "ul" ponemos "ol".

Viñietas

Sirve para redactar ideas o cosas de forma ordenara o desordenada.
Ejemplos:

  • Queso

Ciudades con mayor población de España
