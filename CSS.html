Seleccionando por id(#)

Esta línea se mostrara con un color de fondo específico mediante una id llamada ejemplo
Empleamos esto cuando queremos dar estilo sólo a una cosa.
Es decir, si quieres darle color a un párrafo concreto, escribe lo siguiente p id="ejemplo">el texto que escribas /p
y en head, dentro de style escribes #ejemplo{color:green;}.
